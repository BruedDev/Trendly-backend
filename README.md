# Express.js Project

## Giới thiệu
Dự án này là một API backend đơn giản sử dụng Express.js. Nó cung cấp các endpoint cơ bản để xử lý các yêu cầu HTTP và có thể mở rộng để xây dựng các tính năng phức tạp hơn.

## Yêu cầu
- Node.js (v14.x hoặc mới hơn)
- npm (hoặc yarn)
